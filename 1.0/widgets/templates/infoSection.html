Baltic Sea Impact Index tool (BSII tool)
With the BSII tool it is possible to calculate the Baltic Sea Impact Index and the Baltic Sea Pressure Index, by using the 18 aggregated pressure layers and the 36 ecosystem component layers used in the State of the Baltic Sea report (HOLAS II). The tool uses the data layers, as well as a sensitivity scores matrix as input, and creates a BSPI or BSII grid layer as output.
Baltic Sea Impact Index
Baltic Sea Impact Index calculates the spatial distribution of cumulative impacts on the Baltic Sea. The BSII builds on concepts developed by Halpern et al. (2008), and was first applied in the initial HELCOM holistic assessment. Baltic Sea Pressure Index
Baltic Sea Pressure Index calculates the cumulative pressures of human activities to the Baltic Sea. The BSPI uses the same aggregated pressure layers than BSII, but doesn’t consider the ecosystem components in the calculation. However, the pressures are weighted based on the average sensitivity score of all ecosystem components to each aggregated pressure layers. This is done to avoid overestimation of pressures with large extent, but low effect to ecosystem. Weighting is inbuilt in the tool, and can not be changed in the web tool.
The BSPI is also based on the concept based on Halpern et al. in 2008. How to use the tool
“Layers” tab
Contains the pressure and ecosystem components spatial layers available for the calculation. These layers comprise of the data that were used in BSII for the State of Baltic Sea report (2018). Next to each layer there is a metadata button which redirect to HELCOM Metadata catalogue, where one can read more information about the layer and download it.
By selecting individual layers one can view them on the map. When layer(s) are added to the map one can click specific map location and get layer's value at that location.
“Calculation” tab
One can choose to calculate BSII or BSPI. Then choose layers to include in the calculation.
For the BSII, after selecting the layers and by pressing “Get Sensitivity Scores”, the predefined sensitivity score matrix appears below, and the sensitivity scores can also be edited through this matrix. By pressing “Calculate BSII” the tool will calculate the result BSII raster and BSII statistics matrix.
For the BSPI, after selecting the layers and by pressing “Calculate BSPI” the tool will calculate the result BSPI raster.
Calculated result layers will appear on the map pane on the right side. In the map top right corner a box with layer's legend, download links and map tools will appear. By clicking specific map location one can get layer's value at that location.
BSII and BSPI layers on the map always appear on top of the individual ecosystem component and pressure layers (if those are selected). By using “Show BSII/BSPI layer on top” tool one can change order the layers are displayed. By using “Swipe top layer” tool one can enable/disable swipe slider, which allows to view the layer below the top one.
Metadata and further reading
The metadata descriptions of the Aggregated pressure layers and ecosystem components can be accessed through HELCOM metadata catalogue and the map layers can be viewed at HELCOM Map and Data Service. More information about the tool, methodology and data used can be found at the State of the Baltic Sea report website and from the thematic report on cumulative impacts. Please note: All descriptions of results, layers and sensitivity scores are described as they were used in the Baltic Sea Impact Index produced for the State of the Baltic Sea report (2018).
Tool was developed within Pan Baltic Scope project. Desktop version of the tool, a toolbox for ArcGIS Pro software, includes more tools than BSII and BSPI and can be downloaded from HELCOM GitHub account.
